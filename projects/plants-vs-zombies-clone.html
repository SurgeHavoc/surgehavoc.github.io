Plants vs. Zombies Clone

Tools: Custom Engine, C++, JSON

Overview

Project Description

I collaborated on a team of five to implement a Plants vs. Zombies-style clone in a custom engine using C++ and JSON. My main focus was the UI layer and making it communicate cleanly with gameplay systems.
